## 🎨 Color Scheme

```
PRIMARY COLORS:
├─ Olive Green: #556B2F (Main buttons, borders)
├─ Light Olive: #6B8E23 (Hover states)
├─ Bright Green: #7CFC00 (Text highlights, glow)
├─ Dark Background: #1a1a1a (Page background)
└─ Card Background: #2a2a2a (Card/modal background)

TEXT COLORS:
├─ Primary Text: #C0C0C0 (Main text)
├─ Secondary Text: #808080 (Muted text)
├─ Admin Badge: #ff9800 (Orange for admin)
└─ Error: #ff6b6b (Red for errors)

INTERACTIVE:
├─ Hover: Glow effect (0 0 10px rgba(123, 252, 0, 0.3))
├─ Active: Bright green with underline
└─ Disabled: Opacity 0.5
```

## 🔄 User Roles Flow

```
┌─────────────────────┐
│  VISIT HOMEPAGE     │
│   (Public View)     │
└──────────┬──────────┘
           │
           ├──► [LOGIN] or [REGISTER]
           │      │
           │      └──► Select Role
           │           ├─ STUDENT
           │           └─ ADMIN
           │
           ├─────────────────────────┐
           │                         │
    ┌──────▼─────┐          ┌────────▼──────┐
    │  STUDENT    │          │    ADMIN       │
    └──────┬─────┘          └────────┬───────┘
           │                         │
      Can:                      Can:
      • Create posts       • Delete any post
      • Comment           • Delete comments
      • Like posts        • Ban/unban users
      • View profiles     • View analytics
      • Edit own posts    • Delete users
      • Delete own posts  • Moderate content
           │                         │
           └────────┬────────────────┘
                    │
            [LOGOUT/SIGN-OUT]
                    │
            ┌───────▼────────┐
            │ Back to Login   │
            └─────────────────┘
```
